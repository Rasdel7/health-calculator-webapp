# Personal Health Calculator 💪

Calculates BMI, daily calories, macros, water intake
and simulates weight progress over time.

## Live Demo
[Click here](https://health-calculator-webapp-mncnzvlng3jxcshywqp4kp.streamlit.app)

## Features
- BMI calculator with visual gauge
- Calorie needs using Mifflin-St Jeor formula
- Macro breakdown (protein, carbs, fat)
- Daily hydration schedule
- Weight progress simulator up to 52 weeks

## Tools Used
- Python, Streamlit, Pandas, Matplotlib, NumPy

## How to Run Locally
pip install streamlit pandas numpy matplotlib
streamlit run app.py
